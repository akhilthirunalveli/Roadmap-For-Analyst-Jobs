About

This roadmap covers Excel, SQL, Python, Power BI, and Tableau with two case studies and a final portfolio project.

  • Master advanced Excel functions: VLOOKUP, INDEX/MATCH, PivotTables, conditional formatting.
  • Learn foundational SQL commands: SELECT, FROM, WHERE, GROUP BY, ORDER BY, LIMIT.
  • Practice basic SQL queries on a public dataset.
  • Get comfortable with the pandas library in Python.
  • Practice reading data into a DataFrame from CSVs or Excel.
  • Learn basic data manipulation with pandas: filtering, sorting, handling missing values.
  • Master SQL JOINs: INNER, LEFT, and RIGHT JOINs to combine tables.
  • Practice cleaning messy Excel datasets (removing duplicates, correcting errors, splitting text).
  • Use pandas to handle complex cleaning tasks: converting data types, handling outliers, standardizing text.
  • Utilize SQL's CASE statements and string functions for in-database data cleaning.
  • Download and install Power BI Desktop.
  • Learn the user interface and connect to a simple dataset.
  • Practice using Power Query for data transformation.
  • Understand the concept of a Star Schema for data modeling.
  • Download and install Tableau Public.
  • Learn the user interface and connect to a dataset.
  • Create basic visualizations: bar charts, line graphs, pie charts.
  • Understand the difference between Dimensions and Measures.
  • Create interactive Power BI dashboards.
  • Use slicers and filters to create a data story.
  • Learn simple DAX for creating new columns and measures.
  • Build a multi-visualization dashboard in Tableau.
  • Use actions and filters to make the dashboard interactive.
  • Create a 'story' from your dashboard to present insights.
  • Find a simple public dataset from Kaggle or data.gov.
  • Use Excel and SQL to clean and prepare the data.
  • Create a basic dashboard in Power BI to present key insights.
  • Find a slightly more complex dataset.
  • Use Python (pandas) for the initial data cleaning and manipulation.
  • Build a detailed, interactive dashboard in Tableau.
  • Find a new dataset and go through the full workflow: SQL, Python, and a final dashboard in either Power BI or Tableau.
  • This project will be your portfolio piece.
  • Review all projects and practice explaining your process and findings for interviews.

Progress is saved locally in your browser. Export or sync can be added on request.
